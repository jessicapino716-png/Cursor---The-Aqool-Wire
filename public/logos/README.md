# Company Logos

Place company logo files in this directory with the following naming convention:

- `pif.svg` - PIF logo
- `aramco.svg` - Aramco Ventures logo
- `waed.svg` - Wa'ed Ventures logo
- `stv.svg` - STV logo
- `raed.svg` - RAED Ventures logo

**Supported formats:** SVG (recommended), PNG, JPG

The component will automatically load these logos. If a logo is not found, it will fall back to displaying company initials.
